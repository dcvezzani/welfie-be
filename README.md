# welfie-be

### Basic set up and start
```
npm install
npm start
```

With debug
```
DEBUG=welfie-be.* npm start
```

### Restful endpoints

#### index
```
GET /api/messages


```

#### show
```
GET /api/messages/:id
GET /api/messages/12
```

#### create
```
POST /api/messages
```
